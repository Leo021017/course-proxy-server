## 项目来源

四川大学网络空间安全学院 网络空间安全产品开发实践课程设计项目

**多协议代理服务器**

该系统分为客户端、代理服务器端和后台控制端三个部分，支持HTTP代理、HTTPS代理、以及SOCKS5协议代理，同时提供缓存功能和多线程并发处理，满足用户对网络代理的多样化需求

## 项目目录结构

| client         | 客户端部分       |
| -------------- | ---------------- |
| proxy_backend  | 控制端部分——后端 |
| proxy_frontend | 控制端部分——前端 |
| proxy_server   | 代理服务器部分   |
